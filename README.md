# 💞 Dating Bot — Бот знакомств

## Функционал
- 🌐 **Лента** — просмотр анкет с лайком/дизлайком
- 💞 **Мэтчи** — взаимные симпатии + кнопка "Написать"
- ❤️ **Лайки** — кто тебя лайкнул
- 🔥 **Топ** — рейтинг по количеству лайков
- 👤 **Профиль** — просмотр и редактирование анкеты
- ⚙️ **Настройки** — фильтр по полу, удаление анкеты    
- 🚩 Жалобы на пользователей
- 🔴 Отключение/включение анкеты

## Установка

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Создай бота
- Открой Telegram, найди @BotFather
- Напиши /newbot
- Скопируй токен

### 3. Вставь токен
Открой файл `config.py` и замени:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
```
на твой токен, например:
```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

### 4. Запусти бота
```bash
python bot.py
```

## Структура файлов

## Хостинг (чтобы бот работал 24/7)

### Бесплатно:
- **Railway.app** — бесплатный план, деплой через GitHub
- **Render.com** — бесплатный план
- **PythonAnywhere** — бесплатно для простых ботов

### Платно (надёжнее):
- VPS на **Timeweb** / **Reg.ru** от 150 ₽/мес
- Запуск: `nohup python bot.py &` или через systemd/screen
